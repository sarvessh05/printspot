# PrintSpot — Pure Magic Printing

This is the modern kiosk terminal and landing page for PrintSpot.
Built with Vite, React, Tailwind CSS, and Shadcn UI.

## Features
- ✨ Premium, high-end landing page.
- 📱 Kiosk interface for seamless printing.
- 📊 Admin dashboard for managing kiosks.
- 🌊 Fluid motion with Framer Motion.

## Development
```bash
npm install
npm run dev
```
